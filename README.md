# Boilerplate: Knex.js

Phase 1 boilerplate project with:

 - Express
 - Knex
 - SQLite3
 - Express Handlebars
 - body-parser
 - Jest
 - nodemon


## Install

```
npm install
npx knex migrate:latest
npx knex seed:run
npm run dev
```

Be sure to check out the other npm scripts too.
